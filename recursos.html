Recursos e Referências

Esta página reúne recursos valiosos para aprofundar seus conhecimentos em Programação Orientada a Objetos. Encontre livros, tutoriais, cursos e documentações oficiais.

Livros Recomendados

  • Clean Code: A Handbook of Agile Software Craftsmanship - Robert C. Martin

    Um clássico que explora princípios de bom design orientado a objetos e práticas de código limpo.

  • Head First Design Patterns - Eric Freeman, Elisabeth Robson

    Introdução aos padrões de design orientados a objetos de forma visual e envolvente.

  • Object-Oriented Analysis and Design with Applications - Grady Booch

    Um guia abrangente sobre análise e design orientados a objetos, escrito por um dos criadores da UML.

  • Effective Java - Joshua Bloch

    Guia prático com padrões e melhores práticas para programação orientada a objetos em Java.

  • Design Patterns: Elements of Reusable Object-Oriented Software - Erich Gamma, Richard Helm, Ralph Johnson, John Vlissides (Gang of Four)

    O livro clássico que introduziu os 23 padrões de design mais conhecidos.

Cursos Online

  • Object Oriented Programming in Java (Coursera)

    Curso da Universidade da Califórnia, San Diego, que ensina os princípios de POO usando Java.

  • Object-Oriented Programming (edX)

    Cursos de várias universidades ensinando conceitos de POO em diferentes linguagens.

  • Java: The Complete Object-Oriented Programming Course (Udemy)

    Curso completo que aborda todos os conceitos de POO com exemplos práticos em Java.

  • Learn Python 3 (Codecademy)

    Inclui módulos dedicados a classes e objetos em Python.
